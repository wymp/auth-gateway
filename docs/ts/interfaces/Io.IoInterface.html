Interface IoInterface<ClientRoles, UserRoles>

A complete I/O interface for this service. This allows implementers to pass in a datalayer implementation of their choice.

This interface appears quite dense, but it is actually somewhat simple. It provides basic get, save, update and delete methods for objects in this environment.

  • UserRoles: string
